Practical designs for renewable energy systems

This portfolio presents three engineering design activities prepared for FT-IV: a tidal power plant suitable for a coastal village, a biomass-based biogas system for a 100-cattle dairy farm, and an Ocean Thermal/Geothermal style solution for electricity and freshwater co-production. Each activity contains a block diagram, conversion steps, sizing estimates, and real-world relevance notes.

Course: FT-IV Semester Project

How to use this portfolio

Click the activities in the navigation to view detailed block diagrams and explanations. Images and diagrams are included for clarity — download or view the code via the repository if needed.
